# johnhanacek.com

Portfolio website for John Hanacek, showcasing work at the intersection of **Creativity, Curiosity, AI & Human Augmentation**.

**[www.johnhanacek.com](https://www.johnhanacek.com)**

## About

Product Designer, Consultant, and Artist with an MA in Communication, Culture & Technology from Georgetown University. Awarded "Most Meta" by peers for thesis on AI-enabled drawing apps. Founder Institute SF 2020 graduate. R&D Innovation Award recipient (Aerospace Medical Association, 2022).

## Key Features

- **Interactive Canvas Demos** - Blueprint drawing system (design page)
- **Responsive Design** - Mobile-first layout with dark oceanic aesthetic
- **Performance Optimized** - Smart canvas loops, idle pausing, simplified animations
- **Accessibility** - WCAG AA compliant with keyboard navigation and screen reader support
- **Single-Page Architecture** - Each page is standalone with embedded CSS/JavaScript

## Design System

- **Typography**: Space Grotesk (headings), DM Sans (body), JetBrains Mono (monospace)
- **Colors**: Ink (#1a1a2e), Paper (#f8f6f1), Accent (#e63946)
- **No external dependencies** except Google Fonts
