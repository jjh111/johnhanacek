Brand Identity
Voice & Tone

Terminal-precise, ocean-deep. Technical authority without sterility — every label is lowercase or small-caps mono, headlines are airy Raleway ultralight. The system whispers depth.

Think: ocean floor console. Calm, minimal, luminescent.

Animation Reference
shimmer
Alternates drop-shadow between cyan-glow and gold-glow, opacity 0.95→1. 8s cycle. Used on hero name gradient.
animation: shimmer 8s ease-in-out infinite
bounce
Classic bounce with multi-keyframe: 0→-8px→-4px→0. 2.5s. Scroll/chevron indicators.
animation: bounce 2.5s ease-in-out infinite
bounce-gentle
Soft float: 0→-3px→0. 3s. Used on scroll hint buttons inside hero oval.
animation: bounce-gentle 3s ease-in-out infinite
✏️
pencil-wiggle
Rotate -5deg ↔ 5deg. 4s. Used on the draw-hint icon in the canvas hero.
animation: pencil-wiggle 4s ease-in-out infinite
strokeGlow
SVG stroke transitions from rgba(125,216,247,0.3) to 0.8. Used on shape-nav active states.
animation: strokeGlow 1s ease-in-out infinite alternate

All animations respect prefers-reduced-motion — set to 0.01ms duration in the global reset.
